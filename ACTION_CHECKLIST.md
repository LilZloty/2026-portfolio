## PHASE 1: FOUNDATION (Week 1-4)

### Week 1: Build Your Proof

- [ ] **Day 1-2**: Pick 3 Shopify stores to audit
  - Go to: [shopify.com/examples](https://www.shopify.com/examples) or competitor stores
  - Save their URLs in a doc
  - Priority: Auto parts stores, home goods, any store with visible speed issues

- [ ] **Day 3-5**: Create first video audit
  - Tool: Loom (free tier)
  - Format: 5-8 min walkthrough
  - Cover: PageSpeed score, top 3 issues, how you'd fix them
  - Save video + before screenshots

- [ ] **Day 6-7**: Post and share

## DECISION CHEAT SHEET

### Should I take this opportunity?

```
Does it:
├── Pay money? → How much? Worth my hourly rate?
├── Build proof? → Will I get a testimonial/case study?
├── Build audience? → Will it grow my email list?
└── All NO? → SKIP IT
```

### Should I spend money on this?

```
Will it directly generate revenue in <30 days?
├── YES + under $100 → Buy it
├── YES + $100-500 → Try free version first
└── NO → Don't buy until $5K+/month
```
